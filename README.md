# Qr_Code_Genrator
# 📱 QR Code Scanner & Generator (Flutter)

A simple and clean Flutter application to **scan QR codes** and **generate QR codes** easily.

## 🚀 Features

* 📷 Scan QR Code using camera
* 🔳 Generate QR Code from text or URL
* ⚡ Fast and lightweight
* 🎨 Simple UI

## 🛠️ Technologies Used

* Flutter
* Dart
* qr_flutter package
* flutter_barcode_scanner package

## 📂 Project Structure

lib/
 ├── main.dart
 ├── generate_qr_code.dart
 └── scan_qr_code.dart
